# Conversation Service

대화 서비스 API 프로젝트입니다.

## 개요

이 프로젝트는 대화 관련 AI 기능을 제공하는 서비스입니다.

## 주요 기능

- AI 채팅 기능
- 이미지 생성
- 텍스트 감정 분석
- 음식 계산
- 요약 기능

## 기술 스택

- Python
- Flask (추정)

## 설치 및 실행

```bash
pip install -r requirements.txt
python app.py
```

## Docker 실행

```bash
docker-compose up
```
